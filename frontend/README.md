# AI-agent-TODO Frontend

React + Vite + TypeScript front end for the AI-agent-TODO MVP.

## Development

```bash
npm install
npm run dev -- --port 5173
```

The app reads the API base URL from `VITE_API_BASE_URL` and defaults to:

```text
http://127.0.0.1:8000/api
```

Use `体验演示` on the login page to explore the full UI without a running backend.

## Build

```bash
npm run build
```
