# AI-agent-TODO 后端实现文档

## 1. 文档说明

本文档描述当前 `backend/` 目录中的真实实现，而不是待落地方案。

命名约定：

- 后端实现、仓库和课程项目历史名保留 `AI-agent-TODO`
- 用户可见产品名统一为 `TaskPilot`

当前后端已经实现：

- 用户认证
- 当前用户信息
- 任务 CRUD 与筛选
- AI 解析、AI 创建任务、AI 字段推荐、AI 聊天 Agent
- 用户设置与 BYOK
- 统计分析
- 统一响应和异常处理

## 2. 技术栈

- FastAPI
- SQLAlchemy 2.x
- Alembic
- Pydantic 2
- pydantic-settings
- python-jose
- passlib + bcrypt
- OpenAI Python SDK
- pytest

## 7. 配置与环境变量

配置入口：

- [backend/app/core/config.py](/Users/fiwoouo/Desktop/软工实验/AI-agent-TODO/backend/app/core/config.py:1)

`.env.example` 当前主要字段：

```text
APP_NAME=AI-agent-TODO
APP_ENV=development
DEBUG=true

API_PREFIX=/api
BACKEND_CORS_ORIGINS=http://127.0.0.1:5173,http://localhost:5173

DATABASE_URL=sqlite:///./ai_agent_todo.db

JWT_SECRET_KEY=please-change-this-secret
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=120

API_KEY_ENCRYPTION_SECRET=please-change-this-secret

OPENAI_API_KEY=
OPENAI_DEFAULT_MODEL=deepseek-v4-pro
OPENAI_BASE_URL=https://api.openai.com/v1
DEEPSEEK_BASE_URL=https://api.deepseek.com
DEEPSEEK_RESOLVE_WITH_DOH=true
DEEPSEEK_DOH_URL=https://cloudflare-dns.com/dns-query
DEEPSEEK_FORCE_RESOLVE_IP=
AI_MOCK_MODE=false
```

说明：

- `OPENAI_API_KEY` 是部署级默认 Key，可被用户私有 Key 覆盖
- `OPENAI_DEFAULT_MODEL` 是默认模型名
- `DEEPSEEK_*` 配置用于 DeepSeek 兼容接口和网络解析兜底
- `AI_MOCK_MODE=true` 时，AI 解析和推荐可走本地 Mock 逻辑，方便测试和离线演示

## 8. AI 供应商与模型说明

虽然变量名沿用 `openai_*`，当前实现实际支持两类模型名：

- OpenAI 系列模型，如 `gpt-4o-mini`
- DeepSeek 系列模型，如 `deepseek-v4-pro`、`deepseek-v4-flash`

当前前端默认聊天模型见：

- [frontend/src/features/ai-chat/models.ts](/Users/fiwoouo/Desktop/软工实验/AI-agent-TODO/frontend/src/features/ai-chat/models.ts:1)

后端会根据模型名前缀区分是否走 DeepSeek 兼容 base URL。

## 9. 统一响应与错误处理

相关文件：

- [backend/app/core/response.py](/Users/fiwoouo/Desktop/软工实验/AI-agent-TODO/backend/app/core/response.py:1)
- [backend/app/core/errors.py](/Users/fiwoouo/Desktop/软工实验/AI-agent-TODO/backend/app/core/errors.py:1)

规则：

- 成功响应统一返回 `code/message/data/request_id`
- 删除接口返回 `204 No Content`
- 业务错误通过 `BusinessError` 转换为 JSON
- `422` 校验错误统一包装为 `code = 1001`

## 11. 启动方式

安装依赖：

```bash
cd backend
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
cp .env.example .env
```

初始化数据库：

```bash
.venv/bin/alembic upgrade head
```

启动服务：

```bash
.venv/bin/uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```
